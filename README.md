# 🚀 GitHub Developer Intelligence Platform

Analyze GitHub Profiles, Repositories & Developer Insights Instantly.

A modern React-based web application that fetches and visualizes GitHub user data using the GitHub REST API. Get detailed developer profiles, repository insights, analytics, and technology breakdowns in one place.

---

## 🌐 Live Demo

https://github-developer-intelligence-platf.vercel.app/

---

## ✨ Features

### 👤 Developer Profile
- GitHub Avatar
- Username & Display Name
- Bio
- Followers & Following Count
- Public Repository Count
- Company Information
- Location
- Account Creation Date

### 📊 Developer Analytics
- Total Repositories
- Total Stars
- Total Forks
- Most Used Language
- Top Repository

### 📁 Repository Explorer
- Repository Name
- Description
- Primary Language
- Stars Count
- Fork Count
- Last Updated Date
- Direct Repository Link

### ⚡ Additional Features
- Real-time GitHub User Search
- Loading State
- Error Handling
- Responsive UI
- Modern Dark Theme
- Clean Card-Based Layout

## 🔗 API Endpoints Used

### User Profile

```http
GET https://api.github.com/users/{username}
```

### User Repositories

```http
GET https://api.github.com/users/{username}/repos
```
